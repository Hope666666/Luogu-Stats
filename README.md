### 自定义选项

使用卡片时，支持设定自定义效果选项，可以组合使用。

1. **隐藏标题**，在链接最后带上`&hide_title=true`

2. **黑暗模式**，在链接最后带上`&dark_mode=true`

3. **自定义宽度**，默认 500，限制宽度在 500 到 1920 之间，在链接最后带上`&card_width=blabla...`
